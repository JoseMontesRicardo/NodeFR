# NodeFR
mini framework para [NodeJS](https://github.com/nodejs/node) que usa las caracteristicas de ES6

## Requerimientos
- NodeJS  >=  6.9.1.
- Git >= 2.7.4.


## CLI Interface
* para generar rutas.
```
$ nodefr --generate route [name]
```

* para generar modelos.
```
$ nodefr --generate model [name]
```
